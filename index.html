CityCraft Citybuild

Ein Citybuild Server für die 1.17.1 bzw. 1.17.10 - 1.17.32 auf Bedrock

citycraft.nbb.wtf

Warum dieser Server?

Der Server bietet ein Citybuild System mit Economy, custom Plugins, Rängen und FFA!

Bedrock Support

Der Server supported auch alle Bedrock Version von 1.17.10 - 1.17.32. Um immer auf dem neusten Stand zu sein, guckt bei GeyserMC auf GitHub.

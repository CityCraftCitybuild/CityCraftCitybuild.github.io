Commands

Unten siehst du alle Commands für verschiedene Bereiche.

Plot

/p add SPIELER

Lasse einen Spieler bei dir (ab)bauen, aber nicht, wenn du Offline bist.

/p auto

Erstelle dir ein Plot.

/p claim

Nimm dir das Plot, auf dem du stehst, wenn es keinen Besitzer hat.

/p deny SPIELER

Erlaube einem Spieler nicht, dein Grundstück zu betreten.

/p home

Gehe zu deinem Plot.

/p remove SPIELER

Entferne einen Spieler vom Plot (von Add, Deny, Trust).

/p trust SPIELER

Lasse einen Spieler bei dir (ab)bauen, auch wenn du Offline bist.

/p visit SPIELER

Gehe zu einem Plot von einem Spieler.

Economy

/balancetop

Gibt dir aus, wer die reichsten Spieler sind.

/pay SPIELER

Gib einem Spieler Geld.

/money

Gibt dir aus, wie viel Geld du hast.

Essentials

/msg SPIELER

Schreibe einem Spieler eine private Nachricht.

/r

Schreibe dem zuletzt per MSG geschriebenen Spieler eine MSG, ohne /msg SPIELER zu machen.

/sethome NAME

Setze ein zu Hause..

/home NAME

Gehe zu einem zu Hause.

/delhome NAME

Lösche ein zu Hause.

/tpa NAME

Schicke einem Spieler eine TPA.

/tpaccept NAME

Nimm eine TPA an.

/tpdeny NAME

Lehne eine TPA ab.

/tpahere NAME

Schicke einem Spieler eine TPA zu dir.

/tpacancel

Brich eine TPA ab.

Others

/nav

Öffnet ein Inventar, in dem du dich zu einer Welt teleportieren kannst.

/cb

Gehe in die CityBuild Welt.

/end

Gehe in die End Welt.

/fw

Gehe in die Farmwelt.

/ffa

Gehe in die FFA Welt.

/l

Gehe in die Lobby.

/ne

Gehe in die Nether Welt.
